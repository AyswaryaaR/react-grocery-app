# Grocery Application

## Description
A grocery shopping app built with React and TypeScript, featuring CRUD operations.

## Tech Stack
- React
- TypeScript
- CSS
- JSON Server (for API)
- Git / GitHub

## Features
- Add, edit, delete items (CRUD operations)
- Search and filter functionality
- State management with React hooks

## Learnings
- Learned how to manage component state efficiently with React hooks
- Practiced integrating a API for frontend CRUD operations
